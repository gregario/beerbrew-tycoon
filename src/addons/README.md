# Addons

## GUT — Godot Unit Test Framework

Install GUT before running tests:

1. Open the project in Godot 4
2. Go to AssetLib → search "GUT" → install "Gut - Godot Unit Testing"
   OR manually: download from https://github.com/bitwes/Gut/releases and copy
   the `addons/gut/` folder into this `addons/` directory.
3. Enable the plugin: Project → Project Settings → Plugins → GUT → Enable
4. Run tests: Tools → GUT → Run All

Test files are in `res://tests/`.
